# 🛒 ClickCart

ClickCart is a modern ecommerce web application built using React and Node.js.  
It provides a seamless online shopping experience with secure login, admin access, cart management, and order processing.

## 🚀 Features

- User Registration & Login
- Admin Login & Dashboard
- Add to Cart
- Checkout & Order History
- Role-Based Access Control
- Responsive UI

## 🛠️ Tech Stack

Frontend:
- React.js
- React Router
- CSS

Backend:
- Node.js
- Express.js
- MongoDB

## 📦 Installation

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
cd backend  
npm install  
node server.js

## 🌍 Deployment
Frontend: Vercel  
Backend: Render
